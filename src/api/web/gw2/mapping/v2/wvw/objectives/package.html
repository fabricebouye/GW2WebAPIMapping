This package contains the classes used to map data from the wvw/objectives API v2 endpoint.
More information on this endpoint:
  • Official forum: New endpoint: /v2/wvw/objectives by Lawton Campbell;
  • Wiki: API:2/wvw/objectives
This API is a replacement to the wvw/objective_names.json API v1 endpoint and to the {@code api.web.gw2.mapping.v1.wvw.objectivenames} package.
